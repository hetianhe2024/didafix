# didafix
滴答修是一个基于微服务架构的图片处理平台，采用付费模式，通过调用第三方AI服务提供专业的图片处理功能。项目设计目标是为用户提供简单易用、高质量的图片处理服务。

一个专业的在线图片处理平台，提供图片去水印、智能抠图等功能再运行功能见：https:xiaojingxiu.com

#### 图片处理功能  
- [x] **图片去水印**：智能识别并移除图片水印
- [x] **图片上传**：支持拖拽上传，多格式支持
- [x] **异步处理**：大文件异步处理，状态实时跟踪
- [x] **安全下载**：处理结果安全访问和下载
